# 如何提issues
1.[点击此处](https://github.com/Python-island/Python-island/issues)
2.点击[navbar(导航栏)](/guide/github/functions.html#navbar-导航栏)中的issues,(**如果你打算点图片可以点上面的链接**)
<br>
